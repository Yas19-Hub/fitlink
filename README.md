# 🏋️ FitLink – Gym Management Website

FitLink is a modern gym management website designed to help users explore fitness services, membership plans, trainers, and contact information through an attractive and responsive interface.

## 📌 Features

- 🏠 Responsive Home Page
- 💪 Gym Programs & Services
- 👨‍🏫 Trainer Information
- 💳 Membership Plans
- 📞 Contact Section
- 📱 Mobile-Friendly Design
- 🎨 Modern UI/UX

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
FitLink/
│
├── css/

├── index.html
└── README.md
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Yas19-Hub/fitlink.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No additional installation is required.
